# Linux System Info Script

A Bash script that displays system information on Linux systems. 

## Features
- Hostname and OS version
- Kernel version
- uptime
- CPU information
- Memory usage
- Disk usage
- Logged in users

---

How to Run

chmod +x system_info.sh
./system_info.sh
